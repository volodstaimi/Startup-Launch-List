## ğŸ’¡ Submission Strategy

A smart approach beats blasting everything at once.

### Phase 1 â€” Authority First (Week 1)

Submit to DR 65+ directories that are free. Maximum link equity, zero cost.

> Product Hunt, G2, AlternativeTo, Hacker News, DEV Community, SourceForge

### Phase 2 â€” Paid Boost (Week 2)

Spend $200-500 on high-DR paid listings for enhanced visibility and faster approval.

> Toolify, AI Stage, Tool Pilot, Capterra (especially for B2B SaaS)

### Phase 3 â€” Build Volume (Weeks 2-3)

Submit to 30-50 mid-tier directories. Spread over 2-3 weeks so your backlink profile grows naturally.

> Mix dofollow and nofollow â€” Google expects both.

### Phase 4 â€” Long Tail (Weeks 3-4)

Complete the rest. Even low-DR sites add link diversity and referral traffic.

### âœ… Before You Start

Prepare these once, reuse everywhere:

- [ ] Product name (keep it consistent across all listings)
- [ ] One-liner (10 words max)
- [ ] Full description (150-300 words â€” write 3-4 unique versions to avoid duplicate content flags)
- [ ] Logo (PNG, transparent background, high-res)
- [ ] 3-5 screenshots
- [ ] Pricing info
- [ ] Social links (X/Twitter, LinkedIn, GitHub)
- [ ] Category tags relevant to your product
